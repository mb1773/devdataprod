Project details

Data Input All you need is your birthday. If you were born on July 10, 1969 you would enter 1969-07-10

Data Output Based on an input of 1969-07-10, the program would calculate that you were born on the 5th day of the week (Thursday).

Where:

Sunday - 1, Monday - 2, Tuesday - 3, Wednesday - 4, Thursday - 5, Friday - 6, Saturday - 7
